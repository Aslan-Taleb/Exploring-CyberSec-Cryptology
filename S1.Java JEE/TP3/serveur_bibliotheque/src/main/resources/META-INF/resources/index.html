Serveur de Gestion de Livres

Bienvenue sur notre serveur de gestion de livres. Ce serveur expose plusieurs méthodes pour interagir avec la base de données de livres.

Ajouter un Livre

Utilisez la méthode POST pour ajouter un nouveau livre :

        POST /server/add
        Parameters:
        - titre (String)
        - auteur (String)
        - nbExemplaires (int)

Obtenir la Liste des Livres

Utilisez la méthode GET pour obtenir la liste complète des livres :

        GET /server/list

Rechercher un Livre par Titre

Utilisez la méthode GET avec le paramètre "titre" pour rechercher un livre par son titre :

        GET /server/search?titre={titre}

Mettre à Jour le Nombre d'Exemplaires

Utilisez la méthode POST pour mettre à jour le nombre d'exemplaires d'un livre :

        POST /server/update
        Parameters:
        - titre (String)
        - nbExemplaires (int)

Supprimer un Livre

Utilisez la méthode POST pour supprimer un livre par son titre :

        POST /server/delete
        Parameters:
        - titre (String)

N'oubliez pas de respecter le format des requêtes et de vérifier les réponses du serveur. Pour plus de détails sur l'utilisation de chaque méthode, consultez la documentation du code source.
