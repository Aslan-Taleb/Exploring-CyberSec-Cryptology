Documentation de l'API

GET /api/users

Récupère la liste des utilisateurs de l'annuaire.

GET http://localhost:8080/api/users

POST /api/add

Ajoute un nouvel utilisateur à l'annuaire.

Exemple de requête :

POST http://localhost:8080/api/add Content-Type: application/json { "login": "nouvel_utilisateur", "password": "mot_de_passe" }

POST /api/authenticate

Authentifie un utilisateur avec un login et un mot de passe.

Exemple de requête :

POST http://localhost:8080/api/authenticate Content-Type: application/json { "login": "utilisateur", "password": "mot_de_passe" }

DELETE /api/delete

Supprime un utilisateur de l'annuaire.

Exemple de requête :

DELETE http://localhost:8080/api/delete?login=utilisateur Content-Type: application/json

POST /api/logout

Déconnecte un utilisateur en invalidant sa session.

POST http://localhost:8080/api/logout
